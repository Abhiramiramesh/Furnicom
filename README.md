# Furnicom (Furniture Shop Website)

##  Abstract
Furnicom is a web-based platform for buying and selling furniture, including customized furniture products. The platform allows customers to register, browse, and purchase both standard and customized furniture. Carpenters can view customer orders and add their own customized furniture products. The admin manages the platform by adding carpenters, viewing customers, tracking payments, and overseeing operations. This system streamlines furniture business management and provides an interactive platform for customers and carpenters.



##  Tech Stack
- Frontend: HTML, CSS, Bootstrap, Django Templates  
- Backend:Django, MySQL  



## Modules

### Admin Module
- Add Carpenters  
- View Customers  
- Track Payments  
- Manage Orders  
- View Registration Details  

### Carpenter Module
- View Customized Customer Orders  
- Add Customized Furniture Products  

### Customer Module
- Register and Login  
- Browse Furniture Products  
- Purchase Standard and Customized Furniture  
- View Order History  



##  How to Run
1.Clone the repository: git clone https://github.com/Abhiramiramesh/Furnicom.git
2.Navigate to project directory: cd Furnicom
3.Activate virtual environment: env\Scripts\activate 
4.Install requirements: pip install -r requirements.txt
5.Run Django server: python manage.py runserver
6.Open browser at http://127.0.0.1:8000
